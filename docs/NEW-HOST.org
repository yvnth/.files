#+title: Adding a New Host
#+author: yvnth
#+options: toc:nil

** 1. Create Host Directory

#+begin_src bash
mkdir -p hosts/<hostname>
#+end_src

** 2. Add Host Files

Create the following files under =hosts/<hostname>/=:

- =hardware-configuration.nix= — generated by the NixOS installer; copy it from =/etc/nixos/hardware-configuration.nix=
- =configuration.nix= — system configuration; use =hosts/satella/configuration.nix= as a reference
- =home.nix= — Home Manager configuration; use =hosts/satella/home.nix= as a reference
- =packages.nix= — system packages and fonts; use =hosts/satella/packages.nix= as a reference

** 3. Add to Flake

Add the host under =nixosConfigurations= in =flake.nix=:

#+begin_src nix
nixosConfigurations.<hostname> = nixpkgs.lib.nixosSystem {
  system = "x86_64-linux"; # or aarch64-linux, aarch64-darwin, x86_64-darwin
  specialArgs = {
    inherit inputs;
  };
  modules = [
    ./hosts/<hostname>/configuration.nix
    # ...other modules
  ];
};
#+end_src

** 4. Add to Hosts Table

Add the new host to =hosts/README.org=.

** 5. Create Secrets Directory

#+begin_src bash
mkdir -p secrets/<hostname>/<user>
#+end_src

** 6. Add Age Key to =.sops.yaml=

Generate a new age key on the new host:

#+begin_src bash
sudo mkdir -p /var/lib/sops-nix
sudo age-keygen -o /var/lib/sops-nix/key.txt
sudo chmod 600 /var/lib/sops-nix/key.txt
sudo chown root:root /var/lib/sops-nix/key.txt
#+end_src

Get the public key:

#+begin_src bash
sudo age-keygen -y /var/lib/sops-nix/key.txt
#+end_src

Add the public key to =.sops.yaml=:

#+begin_src yaml
keys:
  - &<hostname>_<username> age1...
creation_rules:
  - path_regex: secrets/<hostname>/.*\.(yaml|json|ini|env|bin)$
    key_groups:
      - age:
          - *<hostname>_<username>
#+end_src

** 7. Build and Switch

#+begin_src bash
sudo nixos-rebuild switch --flake ~/.files#<hostname>
#+end_src
